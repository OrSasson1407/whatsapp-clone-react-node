## üöÄ Features

- **Real-time Messaging:** Instant messaging using Socket.io.
- **User Authentication:** Secure Registration and Login system.
- **Avatar Selection:** Choose an avatar from the DiceBear API.
- **Image Support:** Send and receive images directly in the chat.
- **Online Status:** See who is currently online with a live indicator.
- **Search Contacts:** Filter your contact list instantly.
- **Chat History:** All messages are stored in MongoDB.
- **Emoji Support:** Integrated emoji picker.
